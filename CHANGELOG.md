# Changelog

All notable changes to MoireTracker will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-01-03

### Added
- Initial open source release
- Desktop UI detection with OCR and pattern analysis
- TypeScript/Node.js core with modular architecture
- Python agent system powered by AutoGen
- Electron integration (iframe, webview, programmatic API)
- MCP server for Claude Code integration
- WebSocket real-time communication
- Docker OCR service with Tesseract
- LLM-powered autonomous learning pipeline
- Dynamic category system with auto-approval
- Multi-language support (German/English)

### Features
- Screen capture and element detection
- Text recognition via Tesseract OCR
- CNN-based icon classification
- Continuous learning with RLAF
- Agent-based task automation
- Desktop validation and action execution
- Redis Streams for event processing

### Documentation
- Architecture documentation (German)
- Electron integration guide
- Autonomous pipeline documentation
